🔄 HOW TO USE DIFFERENT MODELS

Option 1: Edit config.py (Recommended)
──────────────────────────────────────
```python
# For best quality (default)
LOCAL_MODEL_NAME = 'sentence-transformers/all-mpnet-base-v2'

# For speed
LOCAL_MODEL_NAME = 'sentence-transformers/all-MiniLM-L6-v2'

# For Q&A systems
LOCAL_MODEL_NAME = 'sentence-transformers/multi-qa-mpnet-base-dot-v1'

# For multilingual
LOCAL_MODEL_NAME = 'sentence-transformers/paraphrase-multilingual-mpnet-base-v2'
```

Option 2: Code Override
──────────────────────────────────────
```python
from embeddings import LocalEmbedding

# Use specific model
gen = LocalEmbedding(
    model_name='sentence-transformers/all-MiniLM-L6-v2'
)
```

Option 3: Environment Variables
──────────────────────────────────────
```bash
export LOCAL_MODEL_NAME=sentence-transformers/all-mpnet-base-v2
export BEDROCK_MODEL_ID=amazon.titan-embed-text-v2
```

⚠️  IMPORTANT: RE-INDEX REQUIRED

If you change models, you MUST re-index your data:

1. Delete existing index:
   ```python
   indexer.create_index(delete_if_exists=True)
   ```

2. Re-index all documents:
   ```python
   indexer.index_documents(your_documents)
   ```

Why? Different models produce different embedding dimensions
and vector representations.
