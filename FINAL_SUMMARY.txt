💡 DISTANCE METRICS GUIDE (AWS Slide 37)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

EUCLIDEAN (L2):
  Formula: d(p,q) = √(Σ(qi-pi)²)
  Range: [0, ∞), lower is more similar
  Use for: Counts, measurements, recommendation systems
  Example: House prices, product features

COSINE SIMILARITY:
  Formula: sim(a,b) = (a·b) / (||a|| · ||b||)
  Range: [-1, 1], 1 = same direction
  Use for: Semantic search, document classification
  Example: Text similarity, news articles

DOT PRODUCT:
  Formula: a·b = ||a|| ||b|| cos(α)
  Range: (-∞, ∞), higher is more similar
  Use for: Collaborative filtering
  Example: Movie ratings, user preferences
